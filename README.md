### 1. Google Authenticator QR Code 生成器
🔐 [立即使用](https://lance70176.github.io/gen_google_secret_key/)

**功能說明：**
- 生成標準 TOTP 雙重驗證 QR Code
- 自動生成隨機 Secret Key
- 支援自訂發行者（Issuer）與用戶名
- 可掃描或手動輸入 Secret Key
- 純前端實現，所有資料都在本地端處理，確保安全性

**使用場景：**
- 為網站或應用程式啟用 2FA 雙重驗證
- 生成 Google Authenticator 相容的驗證碼
- 測試 TOTP 實現
